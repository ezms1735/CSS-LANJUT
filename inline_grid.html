Kesimpulan Skenario 1 adalah: Uji coba ini membuat tata letak grid dengan latar belakang biru dan kotak-kotak item berwarna biru yang lebih muda, menampilkan angka dari 1 hingga 9 dalam tata letak grid.
\Kesimpulan skenario 2 adalah: Pada skenario ini saya menyalin file grid_layout.html menjadi inline_grid.html, kemudian mengubah display menjadi inline grid. Ini digunakan untuk membuat elemen menjadi grid container yang inline. Ini berarti elemen tersebut akan menjadi grid untuk anak-anaknya, tetapi tetap inline dengan elemen-elemen sejajar lainnya. Perbedaan antara display: grid; dan display: inline-grid; terletak pada bagaimana mereka berperilaku dalam hubungannya dengan elemen-elemen sejajar. Keduanya akan berfungsi sebagai grid container untuk anak-anak langsungnya. secara keseluruhan, display: inline grid; digunakan untuk membuat elemen menjadi grid container yang inline, sehingga elemen-elemen dalam grid akan diatur dalam grid secara horizontal dan tetap inline dengan elemen-elemen sejajar lainnya.
