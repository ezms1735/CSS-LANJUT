Keismpulan: Uji coba ini kita menambahkan 3 class yaitu, .grid-container, .item1, .item.
1. Class .grid-container adalah sebauh kelas yang diberikan kepada elem yang akan diatur sebagai grid container.
2. Class .item1 dan class .item adalah sebuah kelas yang diberikan kepada salah satu elemen dalam grid.
Kesimpulan Skenario 1 adalah: Uji coba ini kita menambhakan grid-row: 1/4; pada class item1. Ini digunakan untuk menentukan posisi baris suatu elemen di dalalm grid container class item1, nilai 1/4 ini menunjukkan bahwa elemen tersebut akank mulai dari baris pertama dan terakhir sebelum baris keempat.
Kesimpulan Skenario 2 adalah: Uji coba ini kita menghapus uji coba 1 dan menggantinya menjadi grid-column: 1/4;. Ini digunakan untuk menentukanposisi kolom suatu elemen di dalam grid container class item1, nilai 1/4 ini menunjukkanbahwa elemen tersebut akan mulai dari kolom pertama dan terakhir sebelumkolomm keempat.
